    Install the InDesign Script

    AutoFrameSpecs.jsx has been downloaded. Follow these steps to install it in InDesign.

    1. Open the Scripts panel in InDesign
    Window → Utilities → Scripts (or press Ctrl+Alt+F11 on Windows / Cmd+Opt+F11 on Mac).
    2. Open the User Scripts folder
    In the Scripts panel, right-click on "User" → Reveal in Finder (Mac) or Reveal in Explorer (Windows). Inside that folder, open the Scripts Panel subfolder.
    3. Drop AutoFrameSpecs.jsx into that folder
    Move the downloaded file into the Scripts Panel folder. The Scripts panel in InDesign should update immediately and show AutoFrameSpecs.jsx under "User".
    4. Run the script
    Open a new InDesign document. Double-click AutoFrameSpecs.jsx in the Scripts panel. A file picker will appear — choose the CSV you exported from this tool. The script generates spec sheets automatically.
    Tips
    • The CSV must be one you exported from this tool — other CSVs will not parse correctly.
    • If you also exported PNG frames, the script will look for them in the same folder as the CSV.
    • Update the script anytime by re-downloading: changes to the tool's spec format may require the latest script.
